# disasterfinder

diasterfinder is a Python program for finding the most effective natural disaster to spark online interest.
This program does not advocate natural disasters, however.

## Requirements
### System
- macOS 0.15.7
- PyCharm 2020.2.3
- Python 3.8.5

### Python Libraries
- tensorflow 2.3.1
- numpy 1.18.5
- matplotlib 3.3.3
- python-ta 1.6.0
